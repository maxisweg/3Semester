# 3. Semester Klausurvorbereitung

Zu den drei Klausuren für dieses Semester findet sich jeweils ein Obsidian-Vault:

- Betriebssysteme,
- Rechnernetze,
- Software- und Web-Engineering I.

Ich empfehle, die Vaults einzeln zu öffnen für bessere Übersicht. Alternativ kann ein `.obsidian`-Verzeichnis eines Vaults in den `root` verschoben und dieser dann geöffnet werden.

---
# Aufsetzen

Navigiere zum Dateienpfad, wo du den Obsidian-Vault haben möchtest und führe folgendes aus:

```
git clone https://github.com/maxisweg/3Semester.git
```

Öffne Obsidian und wähle 'Open folder as vault' im Vault-Manager, navigiere zum Dateienpfad, in dem du geklont hast und wähle RS_Klausurvorbereitung aus.

Die verwendeten Fonts sind unter `fonts/` zu finden und können, wenn gewünscht, installiert werden.

Die Obsidan-Vaults beinhalten Theme und Plugins. Diese können unter jeweiligen `.obsidian/themes/` bzw. `.obsidian/plugins` entfernt werden.

Theme: Border (edited via Style Settings)

Plugins:

- Style Settings
- Advanced Canvas

Fonts:

- Mozilla Headline
- Atkinson Hyperlegible Next
- Google Sans Code
